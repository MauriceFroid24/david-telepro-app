# Maurice - Assistant Télépro David

Application Streamlit temporaire pour guider David dans la qualification des rendez-vous énergie.

## Déploiement Streamlit Cloud

1. Envoyer `app.py`, `requirements.txt` et `README.md` à la racine du dépôt GitHub.
2. Sur Streamlit Cloud, sélectionner le dépôt.
3. Main file path : `app.py`.
4. Déployer ou redéployer.

## Important

Streamlit ne conserve pas les données durablement. Après chaque rendez-vous, copier le rapport dans Alltoo et/ou l'envoyer à Maurice.
