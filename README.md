# Assistant Télépro David

Application Streamlit pour qualifier les leads PAC / énergie, scorer les rendez-vous, générer un rapport détaillé et faciliter l'envoi à Simon.

## Lancer en local
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Déploiement Streamlit Cloud
1. Créer un dépôt GitHub.
2. Envoyer `app.py`, `requirements.txt` et `README.md`.
3. Aller sur Streamlit Cloud.
4. New app > choisir le dépôt > fichier principal `app.py`.
5. Déployer.

## iPad / iPhone
Ouvrir le lien Streamlit dans Safari puis cliquer sur Partager > Ajouter à l'écran d'accueil.
